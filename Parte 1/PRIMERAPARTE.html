Primera parte - R: Vuelos de NYC

En este análisis se trabaja con una base de datos que reúne información como detalle de vuelos, información de aerolíneas, información de aeropuertos, datos sobre aeronaves como también datos meterológicos de vuelos que salieron de NYC. El enfoque inicial es identificar patrones para permitir entender mejor qué variables afectan o tienen una mayor relación.

Luego de hacer uso de las librerías y haber llamado la base de datos podemos dar una visualización general de su estructura.

Análisis de la información

Observamos que se tienen cinco tablas:

Mostrar código 
dbListTables(con)
[1] "aerolineas"  "aeropuertos" "aviones"     "clima"       "vuelos"

Comenzando un poco con la inspección de las tablas podemos identificar las conexión que se tiene, esto será utilizado más adelante considerando cada una de las preguntas que se contestarán.

Pregunta 1.

¿Qué aerolínea tuvo el mayor retraso promedio en la salida en 2013?

Consulta con dplyr

Mostrar código 
q1 <- vuelos %>%
  filter(dep_delay > 0) %>%
  group_by(carrier) %>%
  summarise(promedio = mean(dep_delay, na.rm = TRUE)) %>%
  left_join(aerolineas, by = "carrier") %>%
  select(name, promedio) %>%
  arrange(desc(promedio)) %>%
  mutate(rank = row_number())

q1[1:3,1:2]

Consulta con SQL

Mostrar código 
dbGetQuery(con, "
SELECT 
  ae.name,
  AVG(v.dep_delay) AS promedio
FROM vuelos AS v
LEFT JOIN aerolineas AS ae
  ON v.carrier = ae.carrier
WHERE v.dep_delay > 0
GROUP BY ae.name
ORDER BY promedio DESC
LIMIT 3
")

Gráfico:

Mostrar código 
ggplot(q1, aes(x = reorder(name, promedio), y = promedio)) +

Podemos ver que de las dos consultas, obtenemos que la Aerolínea SkyWest Airlines Inc. es la aerolínea con una mayor cantidad de retrasos teniendo diferencias de 5.04721 con la aerolínea Mesa Airlines Inc. y 7.67021 con la aerolínea ExpressJet Airlines Inc..

Pregunta 2.

¿Qué día de la semana tuvo más vuelos retrasados en promedio?

Consulta con dplyr

Mostrar código 
q2 <- vuelos %>%
  mutate(
    fecha = as.Date(paste(year, month, day, sep = "-")),
    dia_semana = weekdays(fecha)
        ) %>%
  filter(dep_delay > 0) %>%
  group_by(dia_semana) %>%
  summarise(
    promedio = mean(dep_delay, na.rm = TRUE),
    total_retrasos = n()
           ) %>%
  arrange(desc(promedio))%>%
  mutate(rank = row_number())

q2[1:3,1:3]

Consulta con SQL

Mostrar código 
dbGetQuery(con, "
SELECT 
  CASE strftime('%w', date(printf('%04d-%02d-%02d', year, month, day)))
    WHEN '0' THEN 'domingo'
    WHEN '1' THEN 'lunes'
    WHEN '2' THEN 'martes'
    WHEN '3' THEN 'miercoles'
    WHEN '4' THEN 'jueves'
    WHEN '5' THEN 'viernes'
    WHEN '6' THEN 'sabado'
  END AS dia_semana,
  AVG(dep_delay) AS promedio,
  COUNT(*) AS total_retrasos
FROM vuelos
WHERE dep_delay > 0
GROUP BY dia_semana
ORDER BY promedio DESC
LIMIT 3
")

Gráfico:

Mostrar código

    data = q2 %>% filter(rank==1),

Los lunes y jueves presentaron los mayores retrasos promedio. Esto podría estar relacionado con una mayor demanda de vuelos y una mayor congestión operativa en esos días, especialmente por viajes laborales y el incremento en el tráfico aéreo. Aunque los viernes también registraron un número considerable de retrasos, el promedio fue ligeramente menor en comparación con los otros días analizados.

Pregunta 3.

¿Cuál es la distribución de los retrasos en la salida para cada aeropuerto?

Consulta con dplyr

Mostrar código 
q3 <- vuelos %>%
  filter(dep_delay > 0) %>%
  group_by(origin) %>%
  summarise(
    promedio = mean(dep_delay, na.rm = TRUE),
    mediana  = median(dep_delay, na.rm = TRUE),
    minimo   = min(dep_delay, na.rm = TRUE),
    maximo   = max(dep_delay, na.rm = TRUE),
    p25      = quantile(dep_delay, 0.25, na.rm = TRUE),
    p75      = quantile(dep_delay, 0.75, na.rm = TRUE),
    desviacion = sd(dep_delay,na.rm = TRUE)
  ) %>%
  left_join(aeropuertos, by = c("origin" = "faa")) %>%
  select(name, promedio, mediana, minimo, maximo, p25, p75,desviacion) %>%
  arrange(desc(promedio))

q3

Consulta con SQL

Mostrar código 
dbGetQuery(con, "
SELECT
    ae.name,
    AVG(v.dep_delay) AS promedio,
    MIN(v.dep_delay) AS minimo,
    MAX(v.dep_delay) AS maximo,
    PERCENT_RANK() OVER (ORDER BY AVG(v.dep_delay) DESC) AS ranking_percentil
    
FROM vuelos AS v
JOIN aeropuertos AS ae
    ON v.origin = ae.faa
    
WHERE v.dep_delay > 0

GROUP BY ae.name

ORDER BY promedio DESC
")

Se analizó la distribución de los retrasos de salida para cada aeropuerto origen, calculando las medidas descriptivas como promedio, mínimo, máximo de retrasos.

El boxplot muestra que los tres aeropuertos presentan una fuere asimetría positiva, la mayoría d elos vueles tienen retrasos relativamente bajos, pero existen algunos casos extremos con retrasos muy elevados. Observandose valores atípicos en los 3 aeropuertos.

La Guardia presenta el mayor retraso promedio, con aproximadamente 41.63 minutos, mientras que Newark Liberty Intl y John F Kennedy Intl registran promedios cercanos a 39 y 38 minutos, respectivamente. No obstante, John F Kennedy Intl presenta algunos de los retrasos máximos más elevados, alcanzando hasta 1301 minutos, lo que evidencia la existencia de casos extremos significativamente altos.

Además, la diferencia entre la mediana y el promedio confirma la presencia de vuelos con retrasos atípicamente altos que incrementan el promedio general. Asimismo, La Guardia y John F Kennedy Intl presentan una mayor dispersión en los retrasos en comparación con Newark Liberty Intl, lo cual puede observarse tanto en el boxplot como en la amplitud de los valores registrados.

Por otro lado, mediante la función PERCENT_RANK() se obtuvo la posición relativa de cada aeropuerto respecto al promedio de retrasos, permitiendo comparar el comportamiento de los aeropuertos dentro de la distribución general y confirmar cuáles presentan mayores niveles de retraso en relación con los demás. Los resultados muestran que La Guardia ocupa la posición con el promedio de retraso más alto dentro del conjunto analizado.

Mostrar código 
q4 <- vuelos %>%
  filter(!is.na(dep_delay),dep_delay>0) %>%
  summarise(
    total = n(),
    retrasados = sum(dep_delay > 30),

  )

q4

Consulta con SQL

Mostrar código 
dbGetQuery(con, "
SELECT 
    COUNT(*) AS total,
    SUM(CASE WHEN dep_delay > 30 THEN 1 ELSE 0 END) AS retrasos,
    (1.0*SUM(CASE WHEN dep_delay > 30 THEN 1 ELSE 0 END) / COUNT(*))*100 AS proporcion
FROM vuelos
WHERE dep_delay IS NOT NULL
AND dep_delay>0
")

Se calculó la proporción de velos encontrnado que aproximadamente el 37.60% de los vuelos presentó retrasos superiores a 30 minutos.
